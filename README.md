#  AI POWERED COMMUNITY BASED MARKETPLACE PLATFORM Project

##  Features
- User Authentication (JWT)
- Buyer: Browse, Cart, Orders
- Seller Dashboard (Add/Edit/Delete Products)
- Admin Product Approval System
- Order Tracking (Placed → Confirmed → Delivered)
- Reviews & Ratings
- Seller Notifications

##  Tech Stack
- Frontend: React, MUI
- Backend: Spring Boot
- Database: MongoDB

##  Status
This project is currently under development and actively being improved.

##  Future Plans
- AI Recommendation System
- Payment Integration
